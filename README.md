# 🚀 Advanced Discord.js V14 TemplateV14 DiscordJS V14 Template 

Here is a template I created using an advanced version of **DiscordJS v14**, designed specifically for you. Enjoy! 

---

## ✨ Features
* 📂 **Advanced Handler:** Support for Slash Commands & Events.
* 🛡️ **Owner Only System:** Built-in owner check for sensitive commands.
* ⚙️ **Dotenv Support:** Securely manage your tokens and IDs.
* 🛠️ **Error Handling:** Global catch for command and interaction errors.

---

## 🛠️ Installation

### 1️⃣ Clone the repository
```bash
git clone https://github.com/furkibuu/
cd discordjs-v14-template
```

### 2️⃣ Upload the packages
```bash
npm install
```

### 3️⃣ Configure `.env`

Create a file named .env and fill in your credentials:

```env

Token=TOKEN_HERE
App_ID=APP_ID_HERE
Owner_ID=YOUR_DISCORD_USER_ID

```

### 4️⃣ Start

```
start.bat
```
